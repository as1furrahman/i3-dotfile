# Keybindings Reference

**Mod Key:** Super (Windows key)

## Core

| Keybinding | Action |
|------------|--------|
| `Mod+Return` | Terminal (Alacritty) |
| `Mod+Shift+Return` | Floating Terminal |
| `Mod+Shift+q` | Close Window |
| `Mod+Shift+c` | Reload i3 Config |
| `Mod+Shift+r` | Restart i3 |
| `Mod+Shift+e` | Exit i3 |
| `Mod+Ctrl+l` | Lock Screen |

## Navigation

| Keybinding | Action |
|------------|--------|
| `Mod+h/j/k/l` | Focus left/down/up/right |
| `Mod+Arrow` | Focus (arrow keys) |
| `Mod+Shift+h/j/k/l` | Move window |
| `Mod+1-0` | Switch workspace |
| `Mod+Shift+1-0` | Move to workspace |

## Layout

| Keybinding | Action |
|------------|--------|
| `Mod+f` | Fullscreen |
| `Mod+s` | Stacking layout |
| `Mod+g` | Tabbed layout |
| `Mod+e` | Toggle split |
| `Mod+v` | Split vertical |
| `Mod+;` | Split horizontal |
| `Mod+Shift+Space` | Toggle floating |
| `Mod+r` | Resize mode |

## Applications

| Keybinding | Action |
|------------|--------|
| `Mod+d` | Rofi launcher |
| `Mod+w` | Browser (Zen) |
| `Mod+n` | File manager (lf) |
| `Mod+Shift+f` | File manager (Thunar) |
| `Mod+Shift+m` | System monitor (btop) |
| `Mod+c` | Camera (cheese) |
| `Mod+F12` | AI Assistant |

## Hardware (Zenbook)

| Keybinding | Action |
|------------|--------|
| `XF86Audio*` | Volume up/down/mute |
| `XF86MonBrightness*` | Screen brightness |
| `XF86KbdBrightness*` | Keyboard backlight |
| `XF86TouchpadToggle` | Toggle touchpad |
| `XF86WebCam` | Toggle webcam |
| `Print` | Screenshot (select) |
| `Shift+Print` | Screenshot (full) |

## Config Editing

| Keybinding | Action |
|------------|--------|
| `Mod+F1` | Edit i3 config |
| `Mod+F2` | Edit alacritty config |
| `Mod+F3` | Edit zshrc |
