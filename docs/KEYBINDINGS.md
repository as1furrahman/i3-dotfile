# Keyboard Shortcuts

| Keybinding | Action |
|------------|--------|
| **Core** | |
| `Mod+Return` | Open Terminal (Alacritty) |
| `Mod+Shift+Return` | Open Floating Terminal |
| `Mod+Shift+e` | **Power Menu** (Lock/Suspend/Reboot/etc) |
| `Mod+l` | **Lock Screen** |
| `Mod+c` | Close Window |
| `Mod+Shift+c` | Reload i3 Config |
| `Mod+Shift+r` | Restart i3 |
| **Navigation** | |
| `Mod+j` / `Mod+Left` | Focus Left |
| `Mod+k` / `Mod+Down` | Focus Down |
| `Mod+b` / `Mod+Up` | Focus Up |
| `Mod+o` / `Mod+Right` | Focus Right |
| `Mod+Shift+arrows` | Move Window |
| `Mod+1-0` | Switch Workspace |
| `Mod+Shift+1-0` | Move Window to Workspace |
| **Apps** | |
| `F9` | App Launcher (Rofi) |
| `F10` | Window Switcher |
| `Mod+n` | File Manager (lf) |
| `Mod+Shift+f` | File Manager (Thunar) |
| `Mod+Shift+b` | Browser (Firefox) |
| `Mod+p` | Password Manager (pass) |
| `Print` | Screenshot |
| **Hardware** | |
| `VolUp/Down` | Volume Control |
| `Mute` | Toggle Mute |
| `BrightUp/Down` | Brightness Control |
